Simple Typed Mapping
     
     This would be loaded into a given programming language environment as an instance object of a Soldier class:
         {
             "!": "Soldier",
             "name": "Benjamin",
             "rank": "Private",
             "serial number": 123456789
         }
     
     Equivalent YAML:
         --- !Soldier
         name: Benjamin
         rank: Private
         serial number: 123456789
